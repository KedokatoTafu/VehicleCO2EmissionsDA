# VehicleCO2EmissionsDA
A comprehensive data analysis and machine learning project focused on analyzing and predicting CO2 emissions from vehicles. This project explores automobile data to identify how factors such as engine size, fuel consumption, and vehicle type impact carbon emissions.

I developed this project to strengthen my skills in data analysis, data preprocessing, and machine learning, while gaining hands-on experience in working with real-world datasets and building data-driven solutions. I also want to support environmental awareness and provide insights that can help reduce vehicle emissions.

## 🛠 Technologies
* Python

* Pandas

* NumPy

* Matplotlib / Seaborn

* Scikit-learn

* Jupyter Notebook / VS Code

## ✨ Features
📊 **Data Analysis**
* **Data Cleaning:** Handle missing values, duplicates, and inconsistent formats

* **Exploratory Data Analysis (EDA):** Identify trends and relationships in CO2 emissions

* **Data Visualization:** Charts and graphs to illustrate emission patterns over time

🤖 **Machine Learning**
* **Model Building:** Train regression models to predict CO2 emissions

* **Random Forest Regressor:** Applied for accurate prediction

* **Model Evaluation:** Measure performance using metrics like MAE, MSE, R²

## 🚀 The Process
**Data Preparation**
* Loaded and explored the dataset
* Cleaned and transformed data for analysis

**Exploratory Analysis**
* Visualized emission trends across years and regions
* Identified key influencing factors
**Model Development**
* Split dataset into training and testing sets
* Trained machine learning models
* Evaluated model performance and optimized results
